About Maze Runner

Maze Runner is an interactive visualization tool designed to demonstrate how pathfinding algorithms work in real-time. It helps users understand complex algorithms through intuitive animations and hands-on interaction.

This project showcases core concepts of Data Structures & Algorithms, including graph traversal, shortest path algorithms, and maze generation.

Built as a solo project to strengthen problem-solving skills, UI design, and system thinking.

⚛️ Frontend

HTML, CSS, JavaScript, React (planned)

🧠 Algorithms

BFS, DFS, Dijkstra, A*

🎯 Goal

Make algorithm learning interactive and visual
